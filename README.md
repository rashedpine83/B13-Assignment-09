# TutorFlow

TutorFlow is a modern online tutor booking platform where students can explore tutors, search by subject, and book personalized learning sessions easily. The platform provides a smooth user experience with authentication, tutor management, booking features, and responsive UI design.

---

## Features

- User Authentication & Authorization
- Browse Professional Tutors
- Search Tutors by Subject
- Book Learning Sessions
- Tutor Pricing & Availability
- Light & Dark Mode Support
- Fully Responsive Design
- Fast Performance with Next.js
- Modern UI using Tailwind CSS & HeroUI
- Dynamic Data Fetching
